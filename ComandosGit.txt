_____ Comando Git ______

git clone ...
	add caminho ssh do repositório no git
	clona o repositório

git status
	mostra os arquivos alterados

git add *
	inclui todos os arquivos alterados para serem upados

git commit -m "..."
	add comentário e as alterações foram salvas

git push
	sobe as alterações pra nuvem

git pull
	baixa todas as atualizações da nuvem
